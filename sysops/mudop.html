Each MudOp command is individually configurable. The MudOps on your system may or may not have access to the commands listed below.

Command What It Does
Sys God (user) Add Life Add's a life to a specific user.
Sys God (user) Clear Suicide Reset a user's suicide password.
Sys God (user) Add Evil [#] Add's certain # of evil.
Sys God (user) Speak [Message] Sends a message from the Gods.
Sys God (user) Change Name [New Name] Changes user name.
Sys God (user) Change Sex

Changes character's gender.
Sys God (user) Enable/Diable Topten Add/Take off a user from the top list.
Sys God (user) Inventory Show a user's inventory.
Sys God (user) Profile Shows a users profile.
Sys God (user) Powers/Spells Shows what spells/powers a user currently has.
Sys God (user) Saint/Good/Neutral Change a users alignment.
Sys God (user) Lawful/Unlawful Flag a user lawful or unlawful.
Sys God (user) Retrain Makes user retrain, gives all CP that should have to level.
Sys God (user) Jail

Sends a user to Jail, duration based on alignment.
Sys God (user) Abilities, Sys God (user) Verify Abilities Shows a users abilities, or a users quest abilities.
Sys God (user) Status Shows the users stats.
Sys God (user) Profile Shows a users profile.
Sys Stat (user) or [Room #] Shows users stats or room stats.
Sys Light (user) Strikes user with a "lightning bolt".
Sys List Users/Evil/Gang/Limited Items List limited items, gang members, or user details.
Sys Disband [Gang Name], Sys Disable/Enable [Gang Name] Disband a gang or Disable them from the top list.
Sys Report Users/Notopten

List all mud players and BBS handles, and players not listed in the top10.
Sys Goto [Silvermere, Support, Newhaven, Rhudaur, Khazarad] Warps a Sysop to desired area.
Sys Configure Show/Maxusers

Shows the current settings or allows you to change the max number of users allowed in at one time.
Sys Map Map of the current area.
Sys Buffers Statictics for the memory buffers.
Sys Initiliaze Re-initiliaze MajorMUD Buffers
Sys Configure Configure runtime perimiters.
Sys Reset Reset monster regeneration.
Sys Reload Reload settings of wccmmud.ini
Defunct User Check Check for inactive players.
Change defunct limits Change limits for being "inactive".
Apply a Module/Update Apply a module and/or update.
Edit Sysop Bulletin Edit MajorMUD Entrance.
Reindex Checks that items are in the game or regening properly.
