Here are some strategies that people have used when they script a couplecharacters together.
 
Example 1:  Wart the Warrior and Pablo the Priest are scripting together.
 
Setup for Wart the Warrior:
1. Use the AutoHeal to speak a healing phrase (such as HEALWART).
2. Setup your Auto Response Slots with this for reattacking:
2a. Look For:           Pablo says "Re-Attack"
2b. Respond With:     break
 
Setup for Pablo the Priest:
1. Setup your Auto Response Slots with this for Healing:
1a. Look For:           HEALWART
1b. Respond With:     GRHE WART
2. Setup your Auto Response Slots with this for having Wart reattack:
2a. Look For:           *Combat Engaged*
2b. Respond With:     Re-Attack
 

Example 2:  Getting the leader tostop when the follower needs to rest.  (Pablo is the Follower.)
 
This method requires several items.
1. Macro Files to go every direction and then pause Path Roaming
2. Leader Path containing the directions that you want to go (thisfile needs to use the Macro Directions)
3. Follower Path containing the command to start the leader movingagain.
4. The following Macros: RESET.MAC,ROAMON.MAC, ROAMOFF.MAC
 
The Leader will need to include the following Auto Responses:
1a. Look For:           Pablo telepaths: roam on
1b. Respond With:     *ROAMON
2a. Look For:           command has no effect.
2b. Respond With:     *RESET
3a. Look For:           Pablo stops to rest.
3b. Respond With:     rest
                        or     sneak
 
Pablo will need to include the following Auto Responses:
1a. Look For:           command has no effect.
1b. Respond With:     *RESET
 
How to run this setup:
1. Download this party.zip file containingall of the files necessary to run a two-man party.
2. Unzip the file.
3. Put all .mac files into the MAC directory of your MMTerm directory.
4. Put all .pth files into the PATH directory of your MMTerm directory.
5. Have the Leader of the group start up the Leader path file first.
6. Have the Follower of the group start up the Follower path file next.
7. You should be all set now.
 
NOTE: This is a slow moving method because the Path File is constantlypausing and resuming.  However, it has been reported that a gain ofabout 8kph has been made because of the slowdown.  It seems that somemonsters will come into a room if a person stays in there for a few seconds.
